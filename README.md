1. Интерфейс запуска 
1) Кнопки выбора сложности (размер слова)
2) Кнопка запуска игры
3) Строка label с названием и правилами игры
2.Интерфейс игры
1) DataGridView размером 5 на 5
2) Две кнопки. Одна проверяет введенное пользователем слово, другая начинает игру сначала.
3) Две строки label. В одной отображается количество “быков”, в другой количество “коров”.
Если буква на правильной позиции, то поле становится зеленого цвета. Если буква присутствует в загаданном слове, но стоит не на своем месте - желтого.
3. Программа
1) InitializeDataGridView()  - насторойка таблицы для ввода
2) Класс GameLogic() - содержит всю игровую логику, отделённую от интерфейса
3) Класс GameResult (связанный вспомогательный класс) - хранит результат одной проверки для последующего отображения или сохранения в файл.
