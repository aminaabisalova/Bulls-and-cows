1. Интерфейс запуска 
Кнопки выбора сложности (размер слова)
Кнопка запуска игры
Строка label с названием и правилами игры
2.Интерфейс игры
DataGridView размером 5 на 5
Две кнопки. Одна проверяет введенное пользователем слово, другая начинает игру сначала.
Две строки label. В одной отображается количество “быков”, в другой количество “коров”.
Если буква на правильной позиции, то поле становится зеленого цвета. Если буква присутствует в загаданном слове, но стоит не на своем месте - желтого.
3. Программа
InitializeDataGridView()  - насторойка таблицы для ввода
2.1 класс GameLogic() - содержит всю игровую логику, отделённую от интерфейса
2.2 Класс GameResult (связанный вспомогательный класс) - хранит результат одной проверки для последующего отображения или сохранения в файл.
